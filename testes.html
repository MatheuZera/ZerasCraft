Acordeão (Accordion)

O que é um Acordeão?

O acordeão é um componente de interface que permite exibir e ocultar seções de conteúdo, economizando espaço na tela. É ideal para perguntas e respostas (FAQ) ou para mostrar detalhes de um produto.

Como usar este Acordeão?

Você pode adicionar quantos "accordion-item" precisar. O JavaScript cuida da lógica de abrir e fechar, adicionando e removendo a classe "active" quando o cabeçalho é clicado.
