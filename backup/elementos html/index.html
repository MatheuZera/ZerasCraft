Como faço para me conectar ao servidor?

Você pode se conectar usando o IP play.zerascraft.com. Certifique-se de usar a versão 1.20.1 do Minecraft. Para mais detalhes, visite a nossa seção de Acesso.

Quais são as regras principais do servidor?

Nossas regras são baseadas no respeito mútuo. Proibimos griefing, hacks, e comportamento tóxico. Para uma lista completa, verifique o canal #regras no nosso Discord.

Quais versões do Minecraft são suportadas?

Atualmente, suportamos as versões 1.20.x, mas a versão recomendada para a melhor experiência é a 1.20.1.

Regras do Servidor

Bem-vindo ao Mundo Zera's Craft! Para manter a experiência divertida para todos, por favor, siga estas regras:

  • Respeito: Trate todos os jogadores com respeito.
  • Anti-Griefing: Não destrua ou modifique construções de outros jogadores.
  • Hacks e Cheats: O uso de qualquer tipo de hack é estritamente proibido.
